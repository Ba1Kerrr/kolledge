# FitLog - Фитнес трекер

Веб-приложение для отслеживания тренировок, питания и фитнес-прогресса.

## Установка

### Предварительные требования
- Python 3.8 или выше
- pip

### Шаг 1: Клонирование репозитория
1. git clone https://github.com/Ba1Kerrr/kolledge
2. cd fitlog
### Шаг 2: Создание виртуального окружения
3. python -m venv venv
### Шаг 3: Активация виртуального окружения
#### Windows:

4. venv\Scripts\activate

#### macOS/Linux:

4. source venv/bin/activate
### Шаг 4: Установка зависимостей
5. pip install -r requirements.txt
### Шаг 5: Запуск приложения
6. python server.py

Приложение будет доступно по адресу: http://localhost:8000

# Использование
## Регистрация
1. Перейдите на главную страницу http://localhost:8000

2. Нажмите "Войти в аккаунт"

3. Выберите "Регистрация"

4. ### Заполните форму:

1. Email

2. Имя пользователя

3. Пароль (минимум 8 символов)

4. Полное имя (опционально)

## $ход в систему
1. Используйте зарегистрированный email и пароль

2. После успешного входа вы будете перенаправлены в личный кабинет

# Основные функции
## Dashboard
1. После входа откроется панель управления с:

2. Статистикой тренировок за неделю

3. Активными целями

4. Последними тренировками

5. Быстрыми действиями

### Добавление тренировки
#### Нажмите "Добавить тренировку" на панели

### Заполните форму:

1. Название тренировки

2. Дата

3. Длительность (минуты)

4. Заметки

Добавьте упражнения:

2. Название упражнения

3. Категория (грудь, спина, ноги и т.д.)

4. Подходы (повторения, вес, отдых)

### Контроль питания
#### Нажмите "Добавить прием пищи"

- Укажите:

1. Название блюда

2. Тип приема пищи (завтрак, обед, ужин, перекус)

3. Время

4. КБЖУ значения (калории, белки, жиры, углеводы)

### Измерения прогресса
#### Нажмите "Записать измерения"

- Введите данные:

1. Вес (кг)

2. Процент жира

3. Обхваты (грудь, талия, бедра, шея)

## Постановка целей
1. Нажмите "Добавить цель"

Определите:

1. Название цели

2. Тип (вес, сила, выносливость)

3. Текущее значение

4. Целевое значение

5. Срок выполнения

# API Endpoints
## Пользователи
- POST /api/users/register - Регистрация пользователя

- POST /api/users/login - Вход в систему

- GET /api/users/me - Информация о текущем пользователе

## Тренировки
- GET /api/workouts/ - Получить список тренировок

- POST /api/workouts/ - Создать новую тренировку

- GET /api/workouts/{id} - Получить тренировку по ID

- PUT /api/workouts/{id} - Обновить тренировку

- DELETE /api/workouts/{id} - Удалить тренировку

## Питание
- GET /api/meals/ - Получить список приемов пищи

- POST /api/meals/ - Добавить прием пищи

- GET /api/meals/daily/summary - Дневная статистика питания

## Измерения
- GET /api/measurements/ - Получить список измерений

- POST /api/measurements/ - Добавить измерение

- GET /api/measurements/stats/progress - Статистика прогресса

## Цели
- GET /api/goals/ - Получить список целей

- POST /api/goals/ - Создать новую цель

- PATCH /api/goals/{id}/complete - Отметить цель как выполненную

## Статистика
- GET /api/stats/dashboard - Основная статистика

- GET /api/stats/workouts/monthly - Месячная статистика тренировок

- GET /api/stats/nutrition/daily - Дневная статистика питания
